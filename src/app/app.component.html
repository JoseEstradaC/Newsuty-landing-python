Newsuty

Newsuty es una aplicación desarrollada para Android (Kotlin, Firebase).
Esta aplicación te permitirá compartir noticias con la comunidad y poder votarlas.
La aplicación esta disponible en 2 idioContacto, Español e Ingles. Tambien podras cambiar facilmente el lenguaje de las noticias que estas leyendo

Guía

Newsuty es muy fácil de usar una. Una vez instalada la aplicación, puedes iniciar sesión con tu cuenta de Google o crear un con tu correo electrónico. Una vez dentro de la aplicación podrás observar que tienes una lista con las noticias más destacadas, puedes votar Like o Dislike con una pulsación larga sobre la noticia.

Compartir

Puedes compartir una noticia ya sea a través de una foto de una noticia en algún medio tradicional o usando la URL de un periódico digital
